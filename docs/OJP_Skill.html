OJP Experience System

Introduction

OJP Enhanced uses an experience system to allow players to customize their characters' weapons and force powers.  Players earn experience points and use them to "buy" their weapons and/or force powers, and can swap them for new weapons and/or powers anytime (you will have to respawn for the changes to take effect).  Experience points accrue during map rotations, so you don't have to start again every time the server changes maps.

 

Experience Levels

The server administer can set the starting number of experience points using the following cvars:

set g_minExpLevel xx
set g_maxExpLevel xx

XX can be any value from 0 to 100.  Assigning a higher value allows players to choose a greater number (or stronger magnitude) of powers and/or weapons when they join the server.  Lower values means players must earn their experience points as they play.  In either case, players can move and reallocate their experience points into any combination they wish.

 

Force Powers

You must first purchase Force Seeing to access your full array of force powers.  You can mix and match light and dark powers as you please...the only limitation is your number of experience points!
